# 关于本书

## 这本书是什么

《如何从零构建 7×24 小时 AI Agent》是一本拆解真实 AI Agent 平台 Halo 的技术书。

不是入门教程,不造概念。每一章对应一个跑在生产环境的真实模块,每个设计决策都有源码行、失败过的备选方案和明确的取舍理由。

## 适合谁读

- **想造 Agent 平台的开发者**:2-5 年经验,能读 TypeScript,不满足于调 API 拿结果
- **正在构建 AI 工具的独立开发者**:卡在"从能用到好用"阶段,需要生产级问题的解决思路
- **好奇"一个人加 AI 能做到什么程度"的人**:第十三章专门回答这个问题

## 不适合谁读

- 想了解"AI 是什么"的入门者(市面上有更好的书)
- 想看通用 LLM 原理(自注意力、MoE 等)的人
- 想要"最佳实践"清单的人——这个领域变化太快,只讨论"约束如何导致设计"
